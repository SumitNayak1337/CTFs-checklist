# 🚩 CTF Master Cheat Sheet

A comprehensive guide for **Capture the Flag (CTF)** competitions, bug bounty hunting, and offensive security assessments.

## 📋 Contents

This checklist covers essential techniques and tools for security testing:

- **Network Reconnaissance** - Network scanning, service enumeration, and information gathering
- **Web Enumeration** - Directory busting, subdomain discovery, and web application analysis
- **Brute Force Attacks** - Password cracking and credential testing
- And more...

## 🎯 Purpose

This cheat sheet is designed to help security professionals, CTF competitors, and bug bounty hunters quickly reference common commands and techniques for:

- Penetration testing
- Vulnerability assessment
- Bug bounty hunting
- CTF competition preparation

## 📚 How to Use

Each section contains practical commands with examples. Commands are organized by category for easy navigation. Customize commands with your target IP addresses, ports, and wordlists as needed.

## ⚙️ Requirements

- Linux-based system (Kali Linux, Parrot OS, etc.)
- Common security tools installed (nmap, hydra, john, gobuster, etc.)
- Basic networking knowledge
- Proper authorization for testing

## ⚠️ Disclaimer

**Use responsibly and legally.** Only perform security testing on systems you own or have explicit permission to test. Unauthorized access is illegal.
